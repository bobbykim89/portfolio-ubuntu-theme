## ALPACA Quest

::

::abstract

An agentic AI questionnaire that helps users discover career paths and the academic programs to get there, through a series of adaptive, context-aware questions.

::

::text-box{title="Tech Stacks" :display-border="false" :rounded="true" border-color="light-1"}

<div class="grid grid-cols-2 gap-1">
<p>Django Rest Framework</p>
<p>OpenAI</p>
<p>Jina AI Embeddings</p>
<p>Qdrant</p>
<p>Nuxt</p>
<p>TailwindCSS</p>
<p>PostgreSQL</p>
</div>

::

- Built an agentic question flow that generates contextual follow-up questions based on user responses.
- Implemented semantic search using Jina embeddings and Qdrant to match responses to relevant career and program data.
- Designed a full-stack architecture with a Django Rest Framework API backend and Nuxt frontend.
